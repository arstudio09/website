# AR Studio09 — Sitio Web Oficial

Sitio web estático para **AR Studio09**, negocio de carpintería y diseño residencial a medida ubicado en Monterrey, Nuevo León, México.

## 📁 Estructura del proyecto

```
ar-studio09/
├── index.html          # Página principal
├── styles.css          # Estilos globales
├── script.js           # Interactividad (menú, galería, lightbox)
├── .gitignore
├── README.md
└── assets/
    └── images/
        ├── logo.jpg
        ├── portada.jpg
        ├── PortadaAR.png
        └── trabajos/   # Fotos del portafolio (1-13)
```

## 🚀 Ejecutar localmente

```bash
# Con Python 3 (sin instalar nada)
python3 -m http.server 8080

# Abrir en el navegador:
# http://localhost:8080
```

## 🌐 Despliegue en GitHub Pages

1. Sube todos los archivos a un repositorio GitHub
2. Ve a **Settings → Pages**
3. En **Source** selecciona `main` y carpeta `/root`
4. Guarda y el sitio estará disponible en `https://tu-usuario.github.io/ar-studio09`
